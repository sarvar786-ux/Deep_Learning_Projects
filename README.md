# 📌 Overview

This repository contains Deep Learning and Computer Vision projects focused on solving real-world problems in healthcare and agriculture using Artificial Intelligence.

The projects demonstrate practical implementation of CNN architectures, image processing, feature extraction, model evaluation, and AI-powered applications using Python, TensorFlow, OpenCV, and Flask.

---

# 🧠 Projects Included

---

## 🌱 Intelligent Plant Disease Detection

### 📖 Description

A Deep Learning-based system designed to detect and classify plant diseases from leaf images using CNN models and image processing techniques.

### 🚀 Features

- Plant disease classification using CNN
- Image preprocessing & augmentation
- Confusion matrix analysis
- Accuracy optimization
- Automated crop disease prediction

### 📈 Achievements

- Improved baseline accuracy from **7% to 10%**
- Reduced manual crop inspection effort by **20–25%**

### 🛠️ Technologies Used

```bash
Python • TensorFlow • Keras • OpenCV • NumPy • Pandas • Matplotlib
```

---

## 🏥 Medical Image Copyright Protection System

### 📖 Description

A Flask-based secure medical image authentication system using Deep CNN-based feature extraction and Zero Watermarking techniques for copyright protection and tamper detection.

### 🚀 Features

- Secure zero watermark generation
- Medical image authentication
- Tamper detection
- Chaotic encryption using Henon Map
- NC & BER analysis visualization

### 🔐 Core Concepts

```bash
Zero Watermarking • CNN Feature Extraction • DCT • Chaotic Encryption
```

### 🛠️ Technologies Used

```bash
Python • Flask • OpenCV • SQLite • NumPy • Matplotlib • HTML • CSS
```

# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/sarvar786-ux/Deep_Learning_Projects.git
cd Deep_Learning_Projects
```

---

## 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Projects

## Flask Application

```bash
python app.py
```
