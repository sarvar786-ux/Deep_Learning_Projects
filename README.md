# 🚀 Deep Learning Projects Collection

## 📌 Overview

This repository contains a collection of Deep Learning, Machine Learning, Computer Vision, and Generative AI projects developed using Python and modern AI frameworks. The projects focus on solving real-world problems in healthcare, agriculture, image processing, predictive analytics, and intelligent automation.

The repository demonstrates practical implementation of Deep Learning models, CNN architectures, data preprocessing, model evaluation, and AI-powered applications using TensorFlow, Keras, OpenCV, and other modern technologies.

---

## 🧠 Projects Included

### 🌱 Intelligent Plant Disease Detection
A Deep Learning-based system for detecting and classifying plant diseases from leaf images using CNN models and image processing techniques.

### 🏥 AI Doctor Assistant
An offline AI-powered healthcare assistant capable of analyzing medical reports, ECGs, X-rays, and scanned documents using Large Language Models and multimodal AI techniques.

### 🔐 Medical Image Copyright Protection System
A secure medical image authentication system using Deep CNN-based feature extraction and Zero Watermarking for copyright protection and tamper detection.

### 📊 Machine Learning & Predictive Analytics Projects
Projects related to predictive modeling, classification, regression, and business intelligence using Machine Learning algorithms.

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/sarvar786-ux/Deep_Learning_Projects.git
cd Deep_Learning_Projects
```

---

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

Activate the environment:

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Projects

### Run Flask Applications

```bash
python app.py
```

### Run Jupyter Notebook

```bash
jupyter notebook
```
