Robust Zero-Watermarking of Medical Images

A professional and non-intrusive copyright protection framework for medical images using frequency-domain feature extraction and chaos-based encryption techniques.

Abstract

As modern healthcare systems increasingly depend on digital image transmission and storage, ensuring image ownership protection and integrity has become a significant challenge. Conventional watermarking approaches modify image content, which is unsuitable for sensitive medical data.

This project presents a robust zero-watermarking framework that extracts discriminative features from medical images without altering the original content. The extracted features are encrypted using chaotic maps to generate secure watermark signatures.

Experimental evaluations demonstrate strong robustness against common image processing attacks such as noise addition, compression, rotation, and scaling, making the proposed system suitable for real-world healthcare environments.

Security

Chaotic encryption enhances resistance against unauthorized watermark reconstruction.

Robustness

Frequency-domain features ensure stability under various image processing attacks.

Medical Integrity

Original medical images remain completely unchanged, preserving diagnostic quality.
